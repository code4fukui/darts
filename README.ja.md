# darts

HTML Canvas APIを使用して描画される、標準的なダーツボードのシンプルな静的Webベースの可視化ツールです。

## デモ

[darts board](https://code4fukui.github.io/darts/)

## 機能

- 公式ルールに準拠した完全なダーツボードを描画します。
- シングル、ダブル、トリプルの正確なスコアと色分けされたセグメントを表示します。
- 外部依存関係のない、単一の `index.html` ファイルで完結しています。
- 純粋なJavaScriptとHTML Canvasを使用しています。ゲームロジックやインタラクティブな要素のない静的な可視化です。

## 使い方

モダンなWebブラウザで `index.html` ファイルを開くだけです。
