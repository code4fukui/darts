# darts

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple, static web-based visualization of a standard darts board, rendered using the HTML Canvas API.

## Demo

[darts board](https://code4fukui.github.io/darts/)

## Features

- Renders a complete, regulation-style darts board.
- Displays accurate scoring numbers and color-coded segments for singles, doubles, and triples.
- Self-contained in a single `index.html` file with no external dependencies.
- Pure JavaScript and HTML Canvas; this is a static visualization with no game logic or interactivity.

## Usage

Simply open the `index.html` file in any modern web browser.
